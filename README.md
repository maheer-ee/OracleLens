# Oracle Lens

Frontend scaffold built with Vite, React, TypeScript, and Tailwind CSS.

## Scripts

- `npm.cmd run dev` starts the local development server
- `npm.cmd run build` creates a production build
- `npm.cmd run lint` runs ESLint

## Structure

- `src/App.tsx` is the main application entry component
- `src/index.css` contains the global Tailwind import and base styles
